Introduction

What is "The Shell"?

A shell is a command line interface that communicates with the computer by taking commands from the keyboard and passing them to the operating system to perform. Most Linux systems have a program called bash (which stands for Bourne Again SHell, written by Steve Bourne) that acts as the shell program.
A terminal is a program that opens a window and lets you interact with the shell. There are windows and linux terminals which are usually launched from the menu on your computer.

Shell programming can be accomplished by directly executing shell commands at the shell prompt or by storing them in the order of execution, in a text file, called a shell script, and then executing the shell script.

Navigation

Files in Unix like operating systems are organised in a tree-like pattern of directories(folders) and subdirectories.
Below are some commands that developers can use to navigate the shell:

  • pwd
  • ls
  • cd

pwd (print working directory)

The directory we are standing in is called the working directory. To see the name of the working directory, we use the pwd command. 

[me@reinepc me]$ pwd 
/home/me

ls (list)

To list the files in the working directory, we use the ls command. 

[me@reinepc me]$  ls 
Desktop    Downloads         foo.txt  Pictures  Templates
Documents  examples.desktop  Music    Public    Videos

File names that begin wuth a (.) character are hidden. To view hidden files .

cd (change directory)

Changing directories can be done by typing cd followed by the pathname of the working directory we want to move to.

Iff we type cd followed by nothing, the command would change the working directory to our home directory.

To change directory, we use the cd command.

[me@linuxbox me]$ cd /usr/bin 
[me@linuxbox me]$ pwd 
/usr/bin
